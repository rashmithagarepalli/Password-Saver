# Password-Saver

## 📌 Project Description
This is a secure Python-based Password Manager that helps users store and manage their passwords safely. It uses encryption to protect sensitive data and ensures that stored credentials are not readable without proper decryption.

---

## 🚀 Features
- Add and store passwords securely
- Encrypt stored data using cryptography (Fernet)
- View saved passwords (with master password access)
- Secure local storage using files
- Simple command-line interface

---

## 🛠️ Technologies Used
- Python
- Cryptography (Fernet encryption)
- File Handling (binary storage)
